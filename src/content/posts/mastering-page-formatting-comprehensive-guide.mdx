## Markdown syntax

Markdown syntax provides a straightforward way to structure your text. Familiarize yourself with some commonly used elements:

### Headers



```markdown
# H1
## H2
### H3
#### H4
##### H5
###### H6
```


### Text formatting


```markdown
**Bold text**
*Italic text*
~~Strikethrough text~~
```


### Lists



```markdown
- Item 1
- Item 2
- Item 3
- [ ] Checkbox
- [x] Checked checkbox
```



### Links



```markdown
[Link text](https://www.google.com)
```


### Images



```markdown
![Image alt text](https://www.google.com/images/branding/googlelogo/1x/googlelogo_color_272x92dp.png)
```


### Code blocks

```markdown
`Inline code`
```

```markdown
```javascript
const foo = 'bar';
```
```


### Blockquotes

```markdown
> Blockquote
```

```markdown
>> Nested blockquote
```


### Tables



```markdown
| Header 1 | Header 2 |
| -------- | -------- |
| Cell 1   | Cell 2   |
| Cell 3   | Cell 4   |
```


### Details

```markdown
<details>
This is a details block
</details>
```

```markdown
<details open>
This is a details block that is open by default
</details>
```

```markdown
<details><summary>Details with summary</summary>
This is a details block with a summary
</details>
```


While not all elements may be used frequently, having them at your disposal enhances your flexibility. Embrace Markdown syntax to simplify both the writing and reading processes.

## Custom components

Beyond Markdown, our platform offers custom components to elevate your page formatting.

#### Alert



```javascript

</Alert>
```

- Info: <Alert type="info" message="Hello, World" includeCloseButton={false} />
- Warning: <Alert type="warning" message="Hello, World" includeCloseButton={false} />
- Danger: <Alert type="danger" message="Hello, World" includeCloseButton={false} />
- Success: <Alert type="success" message="Hello, World" includeCloseButton={false} />
